MTK FRP Bypass + Root

mtkclient v2.0.1 dark mode root guide
1. Environment Setup
create & activate venv
$ python3 -m venv .
$ source bin/activate
$ python3 mtk_gui.py
Make sure you're inside the mtkclient directory. The virtual environment keeps dependencies clean.
2. Enter BROM Mode
  • Power off the phone completely
  • Hold Volume Up + Volume Down simultaneously
  • While holding buttons, connect USB cable to PC
  • Device should now be in BROM mode (mtkclient will detect)
You'll need to enter BROM mode multiple times throughout these processes. Keep these steps handy.

FRP BYPASS

3. Unlock Bootloader
$ python3 mtk.py da seccfg unlock

This command unlocks the bootloader (necessary for full FRP bypass on some devices).

4. Remove FRP Partitions
list partitions (find FRP names)
$ python3 mtk.py rl list
erase FRP and related partitions
$ python3 mtk.py e frp
$ python3 mtk.py e protect1
$ python3 mtk.py e protect2
$ python3 mtk.py e persistence # if exists
Note: Erasing only these partitions does NOT wipe your OS or personal data. It only removes Google account lock.
5. Reboot Device
$ python3 mtk.py reset

The phone will restart. You can now go through setup without Google account prompt.

ROOTING GUIDE

Prerequisites
  • ✓ Developer Options enabled (tap build number 7 times)
  • ✓ USB Debugging enabled
  • ✓ OEM Unlocking enabled
  • ✓ Stock firmware backup recommended
Step 1: Unlock Bootloader

Enter BROM mode, then:

$ python3 mtk.py da seccfg unlock
$ python3 mtk.py reset
Step 2: Dump Stock Boot Image

Enter BROM mode again, then dump partitions:

$ python3 mtk.py r boot,vbmeta boot.img,vbmeta.img
$ python3 mtk.py reset
Step 3: Patch with Magisk
transfer boot.img to phone
$ adb push boot.img /sdcard/Download/
On your phone:
  1. Install Magisk APK
  2. Open Magisk → Install → Select and Patch a File
  3. Navigate to /sdcard/Download/boot.img
  4. Tap "Let's Go" — patched file will be saved in Downloads
retrieve patched image
$ adb pull /sdcard/Download/magisk_patched-xxxxx.img boot.patched
Step 4: Flash Patched Image

Enter BROM mode one final time:

disable vbmeta verification (prevents boot loops)
$ python3 mtk.py da vbmeta 3
flash patched boot image
$ python3 mtk.py w boot boot.patched
For A/B slot devices: Use python3 mtk.py w boot_a boot.patched and python3 mtk.py w boot_b boot.patched
Step 5: Reboot & Enjoy
$ python3 mtk.py reset
Root achieved!

Open Magisk app — it should show "Installed" and you'll have root access.

⚠️ Important Notes

• FRP erase does NOT require new firmware — just reboot.
• Rooting requires bootloader to be unlocked first (already done in FRP section).
• Both processes will wipe your data — backup first!
• If you get stuck, re-enter BROM mode and try again.
